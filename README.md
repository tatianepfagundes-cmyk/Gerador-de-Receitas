🍽️ Gerador de Receitas

Um projeto desenvolvido em Python que gera receitas de forma automática com base em ingredientes ou escolhas do usuário. Ideal para quem quer praticidade na cozinha ou explorar novas combinações culinárias!

🚀 Funcionalidades
🧾 Geração automática de receitas
🥕 Sugestão de pratos com base em ingredientes
⏱️ Receitas rápidas e práticas
🎲 Opção de receita aleatória
💡 Interface simples e fácil de usar
🛠️ Tecnologias utilizadas
Python
Tkinter (interface gráfica)
Bibliotecas padrão do Python
